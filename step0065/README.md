Controller mit Spring Boot

- Java 21 JDK

```shell
mvn clean package spring-boot:run
```
Open in browser:
- http://localhost:8080/hello.html
- http://localhost:8080/api/hello?name=Georg
- http://localhost:8080/api/hello/Georg
